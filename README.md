# BancaFlow

Workflow de gestão para operações bancárias - Stack NestJS + React/Next.js

## 🛠️ Stack Tecnológico

- **Backend**: NestJS
- **Frontend**: React / Next.js
- **Styling**: Tailwind CSS
- **ORM**: TypeORM
- **Database**: PostgreSQL
- **Auth**: JWT

## 📁 Estrutura do Projeto

```
BancaFlow/
├── apps/
│   ├── api/          # Backend NestJS
│   └── web/          # Frontend Next.js
├── packages/
│   ├── shared/       # Código compartilhado
│   └── ui/           # Componentes UI reutilizáveis
└── docs/             # Documentação
```

## 🚀 Quick Start

```bash
# Backend
cd apps/api
npm install
npm run dev

# Frontend
cd apps/web
npm install
npm run dev
```
